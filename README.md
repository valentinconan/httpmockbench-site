# HttpMockBench — Public Site & Issue Tracker

This repository serves two purposes:

1. **Public website** hosted via GitHub Pages — landing page and privacy policy for HttpMockBench.
2. **Issue tracker** — bug reports and feature requests from users.

---

## HttpMockBench

A Chrome extension (MV3) combining an API client, a real-time HTTP logger and a request mocker — all in one DevTools-native panel. No proxy, no config, no server.

- **Free tier:** API client (all methods, headers, auth, body), real-time HTTP logger, request mocker (fixed response, status code, delay, redirect), Freeze to Mock, collections (3), environments (3), request history (50), cURL import & export.
- **Pro tier:** unlimited mock rules, unlimited collections, unlimited environments, unlimited history with full-text search. Coming soon: Postman/Insomnia/Bruno import, stateful mocking, dynamic response templates, pre/post request scripts, regex URL matching.

## Reporting a bug

Please use the [Bug Report](.github/ISSUE_TEMPLATE/bug_report.md) template and include:

- Chrome version and OS
- Extension version (visible in `chrome://extensions`)
- Steps to reproduce
- Expected vs actual behavior
- The URL or request involved if relevant (anonymize sensitive data)
